## Interpretation guide

- `transition_accuracy`: local one-step route acquisition.

- `composition_accuracy`: unseen multi-step recurrent traversal from the learned route field.

- `context_bleed`: competing mode assemblies injected into the active context; this should stress inhibition.

- `feedback_noise`: probability that a transition target is corrupted during training; this should stress reward gating.

- `reward_delay_steps`: feedback delay in the one-step stream; this should stress eligibility traces.

- `mean_correct_margin`: target score minus strongest wrong target.

- `mean_context_margin`: correct-mode support minus strongest wrong-mode support for the same target.

- `mean_wrong_route_activation`: bounded proxy for competing route activation.
