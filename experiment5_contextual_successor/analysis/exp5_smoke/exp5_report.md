## Interpretation framework

- `best_composition_accuracy`: whether the graph can compose mode-conditioned transitions across multiple steps.
- `composition/accuracy_mode_*`: whether some contexts are easier or more fragile than others.
- `composition/accuracy_steps_*`: whether longer traversals degrade gracefully or collapse.
- `composition/average_wrong_route_activation`: whether competing routes remain active even when the answer is correct.
- `composition/confusion_*`: whether the final active state is preserving the intended mode or drifting into another route family.
- `exp5_no_context_routing`: tests whether persistent context injection is load-bearing.
- `exp5_no_inhibition`: tests whether suppressing competing contexts reduces route mixing.
